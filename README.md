## Develop:

1.  **Clone the site.**

    Clone the site.

    ```shell
    git clone https://github.com/tuhinpal/tuhinpal.github.io.git
    ```

2.  **Install dependencies & Start developing.**

    Go into your new site's directory and fire it up.

    ```shell
    cd tuhinpal.github.io/
    npm install
    npm run develop
    ```
